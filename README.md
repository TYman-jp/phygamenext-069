# PhyGame Next — 取扱説明書 (User Manual)

「PhyGame Next」は、ブラウザ上で直感的に物理現象を観察・検証できるリアルタイム物理シミュレーターです。
現在、**「光の屈折シミュレーション」** と **「水面波シミュレーション」** の2つの実験モードを搭載しています。

---

## 1. アプリケーションの起動方法

本システムは外部パッケージへの依存がなく、Node.js環境があれば即座に起動できます。

### 起動手順
1. ターミナルでプロジェクトのルートディレクトリに移動します。
2. 以下のいずれかのコマンドを実行します。
   ```bash
   # デフォルト（ポート: 3000）で起動する場合
   npm start
   
   # ポートを指定して起動する場合（例: 8080）
   node server.js 8080
   ```
3. ブラウザを開き、`http://localhost:3000`（または指定したポート番号のURL）にアクセスします。

---

## 2. 画面レイアウト概要

実験室をイメージしたダークなテーマのUIで構成されています。
*   **SIMULATION CANVAS (上部)**: 物理現象がネオンカラーのアニメーションでリアルタイムに描画されるメインスクリーンです。
*   **PARAMETERS (左下パネル)**: パラメータのスライダー調整、プリセット選択、操作ツールの切り替えを行います。
*   **REFRACTION QUIZ (中央下パネル)**: 光の屈折モード時に利用できる、的に光を当てるゲーム感覚の学習コーナーです。
*   **SIMULATION LOG (右下パネル)**: 実行したシミュレーションやクイズのパラメータ履歴を一覧表示し、CSVに書き出します。

## 3. 「光の屈折」モードの使い方

光が異なる媒質に進む際の「屈折」や「全反射」のメカニズムをシミュレートします。

### 3.1 基本的な操作
1. **パラメータの設定**:
   - **入射角**: 光線が境界の法線（垂直線）となす角度を設定します。
   - **媒質1 屈折率 (\(n_1\))**: 光が入射する側の媒質の屈折率を設定します（例: 空気は `1.00`）。
   - **媒質2 屈折率 (\(n_2\))**: 光が屈折して進む側の媒質の屈折率を設定します（例: ガラスは `1.50`）。
2. **プリセットの利用**:
   - スライダーの下にある「空気→水」「空気→ダイヤ」などのボタンを押すと、代表的な物質の組み合わせと角度にワンクリックで同期されます。
3. **スネルの法則の確認**:
   - スライダーを動かすと、スネルの法則の公式「\(n_1 \cdot \sin(\theta_1) = n_2 \cdot \sin(\theta_2)\)」が画面上にリアルタイムで計算・更新表示されます。
4. **シミュレーション実行**:
   - 「▶ シミュレーション実行」ボタンを押すと、光線が伸びるアニメーションが再生され、結果が「SIMULATION LOG」に1件記録されます。

> [!NOTE]
> **全反射 (Total Internal Reflection) について**
> 屈折率の高い媒質から低い媒質（例：ガラスから空気）へ大きな角度で光が入射した際、屈折光がなくなりすべての光が反射する「全反射」が発生します。全反射が発生すると、警告バッジと警告メッセージ（イエロー）が表示されます。

## 4. 「屈折的当てクイズ」の遊び方

物理パラメータを調整して、Canvas上のターゲット（黄色の的）に屈折光を命中させるモードです。

### 4.1 プレイ手順
1. **問題生成**:
   - 「問題生成」ボタンを押すと、Canvas上に黄色い的（`TARGET`）が出現します。
   - クイズパネルに「媒質条件（例: 空気から水 / 入射角 45.0° 固定）」や「的の中心座標」が提示されます。
2. **ルールの順守**:
   - 「〇〇固定」という条件がある場合、該当するスライダーの値は問題生成時に自動で同期されます。この値を動かさずに、**残りのスライダーのみを調整**して的を狙ってください。
3. **回答判定**:
   - 光線が的に当たっていると思ったら「回答判定」を押します。
   - 判定ロジックにより、光線と的の中心距離が計算され、円内を通っていれば「正解（グリーン）」、外れていれば「不正解（イエロー）」が表示され、ログに自動記録されます。
   - 固定条件の値を崩した状態で判定を押すと、「ルール違反」として不正解扱いになります。
4. **答え表示**:
   - どうしても当たらない場合は、「答え表示」を押すことで、命中するパラメータの一例（唯一解ではありません）をテキストで確認できます。

## 5. 「水面波」モードの使い方

水面を上から見下ろした状態で、波の伝播や干渉、障害物を回り込む「回折」などの波動現象を観察できます。

### 5.1 基本的な操作
1. **モードの切り替え**:
   - 画面最上部ヘッダーの「水面波」タブをクリックします。レイアウトが自動的に水面波モードに切り替わります。
2. **波パラメータの設定**:
   - **波の高さ（振幅）**: 発生させる波のエネルギー（波高）を設定します。
   - **波の速さ（伝播速度）**: 波が水面を伝わる速度を設定します。
   - **周波数（発生頻度）**: 波源から波を送り出す振動数（Hz）を設定します。
   - **液体の粘度（減衰）**: 水のさらさら度/どろどろ度を設定します。粘度が高い（どろどろ）ほど波はすぐに消え、低い（さらさら）ほど波は遠くまで伝わり、反射波が残りやすくなります。
3. **シミュレーション実行**:
   - スライダーを調整するとリアルタイムに反映されます。「▶ シミュレーション実行」を押すと、波の様子がリセットされて最初から再生され、現在の設定値が「SIMULATION LOG」に記録されます。

### 5.2 障害物の手書きと消去
水の中に「壁（障害物）」を自由に作って、波の跳ね返りや回り込みをテストできます。
*   **障害物ペン**: ボタンをアクティブにし、Canvas上をマウスでドラッグまたはタッチすることで、青色の壁（障害物）を描画します。
*   **消しゴム**: ボタンをアクティブにし、ドラッグすることで描画した障害物を消去できます。

### 5.3 障害物のプリセット
スライダーパネル内のボタンから、波動の代表的な物理実験環境をワンタッチで展開できます。
*   **単一スリット**: 中央に壁を配置し、小さなスリット（隙間）を1つ開けます。隙間を通った波が同心円状に広がる「回折現象」が観察できます。
*   **二重スリット**: 隙間を2つ開けます。2つの隙間から出た波が互いに強め合ったり打ち消し合ったりする「干渉縞（干渉現象）」が観察できます。
*   **円柱障害物**: 波の進む先に円形の障害物を配置し、波が回り込む様子を観察できます。
*   **全消去**: 描画した障害物をすべてクリアします。

## 6. 実行ログの管理・エクスポート

実験を行ったパラメータは、ブラウザの `localStorage` に自動的に保存され、リロードしても消えません（最大100件まで保持）。

### 6.1 ログの見方
ログテーブルには、以下の情報が追加順（最新が一番上）に表示されます。
*   **タイプ**: 「通常（屈折シミュレーション）」、「クイズ（的当て回答）」、「水面波」のいずれであるかを示します。
*   **n1 / 振幅**: 屈折率 \(n_1\) または水面波の振幅。
*   **n2 / 速さ**: 屈折率 \(n_2\) または水面波の伝播速度。
*   **入射角 / 周波数**: 入射角（°）または水面波の周波数（Hz）。
*   **屈折角 / 粘度**: 屈折角（°）または水面波の粘度（減衰率の数値）。
*   **結果**: 「屈折」「全反射」などのシミュレーション結果、または「正解」「不正解」「障害物あり/なし」のステータスが表示されます。

### 6.2 ログの操作
*   **ログのクリア**:
    - 「クリア」ボタンを押すと、確認ダイアログの後にすべてのログ履歴が削除され、`localStorage` からも消去されます。
*   **CSVエクスポート**:
    - 「CSV」ボタンを押すと、現在の表示に基づいた履歴データが `phygame_log_YYYYMMDD_HHMM.csv` のファイル名でダウンロードされます。
    - CSVファイルには文字化け対策用のBOM（Byte Order Mark）が付与されているため、Excel等でそのままダブルクリックしても日本語が正しく表示されます。
